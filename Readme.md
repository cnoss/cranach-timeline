## Setup

### Backend

* `cd backend`

* `npm install`

* `npm run import`

* `npm run start`

### Frontend

* `cd frontend`

* Follow [this](https://github.com/BenPag/cranach/tree/develop/frontend) setup guide

* Access the application at http://localhost:8080

### Exit the System
    
Press `Ctrl` + `C`
